# Suggested Screenshots for Submission

Capture the following evidence for the report and appendix:

1. `Home page`
   - show the project overview and rubric-oriented justification

2. `Dashboard`
   - include batch table, statistics, and recent audit feed

3. `Create batch form`
   - show batch metadata fields and wallet/network state

4. `MetaMask transaction confirmation`
   - demonstrate live blockchain interaction

5. `Successful batch detail page`
   - show on-chain batch ID, status, custodian, and timeline

6. `Transfer submission`
   - show a custody transfer request being initiated

7. `Transfer acceptance`
   - switch wallet account and accept the pending transfer

8. `Public verification page`
   - show provenance lookup by batch code or on-chain ID

9. `Swagger/OpenAPI docs`
   - evidence of backend documentation quality

10. `Hardhat test output`
    - evidence of blockchain test coverage

11. `API test output`
    - evidence of backend testing

12. `GitHub repository view`
    - show clean project structure and commit history
